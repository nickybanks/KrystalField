# KrystalField

KrystalField is a mastering-grade dynamic processor featuring dual compression engines for simultaneous downward and upward compression. 

This approach provides precise control over mix dynamics, making it suitable for mastering, bus processing, and final mix polish.

---

## Downloads

All official releases are available in the **Releases** section of this repository.

Each release includes:
- Windows VST3
- macOS VST3
- macOS Audio Unit (.component)

Please download the latest stable version unless you specifically need an older build.

---

## Compatibility

**Windows**
- VST3 (64-bit)

**macOS**
- VST3 (Universal: Intel + Apple Silicon)
- Audio Unit (AU)

All macOS versions are code-signed and notarized.
